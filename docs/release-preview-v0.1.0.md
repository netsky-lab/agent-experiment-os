# v0.1.0 Research Preview

Planned release name:

```text
v0.1.0-research-preview
```

## Included

- MCP server for agent pre-work protocols.
- `agent_work_context.v1` and `agent_presentation_contract.v1`.
- Postgres + pgvector-backed wiki and hybrid retrieval.
- GitHub issue ingestion as evidence-only source and claim pages.
- Matrix runners for Drizzle version traps and Python API drift.
- Run event metrics, churn drill-downs, and completion contracts.
- Policy review gates with evidence ids and rationale.
- Static GitHub Pages thesis homepage.
- Next.js dashboard scaffold for product UI development.

## Not Yet Stable

- Public hosted dashboard.
- Auth and multi-user review workflow.
- Background scheduler for issue refresh.
- Multi-model matrix orchestration beyond current Codex-focused harness.
- Hosted MCP endpoint.

## Release Criteria

- CI green on backend and frontend.
- README quickstart tested from a clean checkout.
- GitHub Pages published.
- At least one public matrix report linked from README.
